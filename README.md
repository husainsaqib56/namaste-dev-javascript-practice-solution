# JavaScript Practice Solutions

This repository contains solutions to various JavaScript practice problems.

## Programs

- [01-Validate Palindrome](01-validate-palindrome.js) - Validates whether a string is a palindrome (ignoring spaces and special characters)
- [02-Flatten Array](02-flatten-array.js) - Flattens nested arrays into a single-level array
- [03-Generate A to Z](03-generateAtoZ.js) - Generates an array containing all letters from A to Z
- [04-Smallest Number in Array](04-smallest-number.js) - Finds the smallest number in an array with input validation
- [05-Count Even Number in Array](05-count-even-numbers.js) - Counts the number of even integers in an array
- [06-Voting Eligiblity](06-voting-eligibility.js) - Checks if a person is eligible to vote based on their age
- [07-Count Negative in Array](07-count-negative.js) - Counts the number of negative numbers in an array
- [08-Find Maximum Sub Array](08-max-subarray.js) - Finds the maximum sum of a contiguous subarray (Kadane's algorithm)
- [09-Check if Power of Two](09-is-power-of-two.js) - Checks if a number is a power of two using bitwise operations
- [10-Sum of All Numbers in Array](10-sub-of-all-numbers.in-in-array.js) - Calculates the sum of all numbers in an array using recursion and reduce
- [11-Largest Number Formed](11-largest-num-formed.js) - Write a function that arranges a list of non-negative integers to form the largest posible number
- [12-Make Counter Function](12-make-counter.js) - Write a function that returns counter object with methods to increment, decrement and reset
- [13-Custom Sort](13-custom-sort.js) - Sorts a mixed array of strings and numbers by separating them and sorting individually, then combining them
- [14-Count Vowels](14-count-vowels.js) - Counts the number of vowels in a string using a Set for efficient lookup

---

**Note:** Each link points to the corresponding `.js` file in the same directory. Click any link to view the solution.
