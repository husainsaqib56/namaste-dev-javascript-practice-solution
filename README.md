# JavaScript Practice Solutions

This repository contains solutions to various JavaScript practice problems.

## Programs

- [01-validate-palindrome.js](01-validate-palindrome.js) - Validates whether a string is a palindrome (ignoring spaces and special characters)
- [02-flatten-array.js](02-flatten-array.js) - Flattens nested arrays into a single-level array
- [03-generateAtoZ.js](03-generateAtoZ.js) - Generates an array containing all letters from A to Z
- [04-smallest-number.js](04-smallest-number.js) - Finds the smallest number in an array with input validation
- [05-count-even-numbers.js](05-count-even-numbers.js) - Counts the number of even integers in an array
- [06-voting-eligibility.js](06-voting-eligibility.js) - Checks if a person is eligible to vote based on their age
- [07-count-negative.js](07-count-negative.js) - Counts the number of negative numbers in an array
- [08-max-subarray.js](08-max-subarray.js) - Finds the maximum sum of a contiguous subarray (Kadane's algorithm)
- [09-is-power-of-two.js](09-is-power-of-two.js) - Checks if a number is a power of two using bitwise operations

---

**Note:** Each link points to the corresponding `.js` file in the same directory. Click any link to view the solution.
